### Elasticsearch 安装

官方网站
> https://www.elastic.co

下载地址
> https://www.elastic.co/downloads/elasticsearch

解压
> unzip elasticsearch-6.0.0.zip

进入根目录
> cd elasticsearch-6.0.0

启动elasticsearch
> bin/elasticsearch

测试启动是否成功
> curl http://localhost:9200/
